## LINUX: 
### [Voltar](#gcc-tutorial-windows--linux)

### Ubuntu / Debian
```bash
sudo apt update
sudo apt install build-essential
``` 

### Fedora
```bash
sudo dnf update
sudo dnf install gcc gcc-c++ make
``` 

### Arch Linux / Manjaro
```bash
sudo pacman -Syu base-devel
``` 

### OpenSuse
```bash
sudo zypper install -t pattern devel_C_C++
``` 

### Para Verificar se instalou digite: 
```bash
gcc --version
```
